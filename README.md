# Game

### Build jar
1. ./gradlew desktop:dist
### Download jar
* [Last version](https://github.com/Programistich/Game/releases)
### Install jar
2. cd %folder_jar%
3. zip -d %name_jar%.jar META-INF/*.RSA META-INF/*.DSA META-INF/*.SF
4. Need install java <12
5. java -jar %name_jar%.jar
